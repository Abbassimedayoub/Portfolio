# Personal Portfolio Website

## Overview

My personal portfolio website, built from scratch with **HTML, CSS and vanilla JavaScript** — no framework, no build step. It presents who I am, my experience, my projects and how to contact me, with a responsive layout that adapts to desktop and mobile screens.

## Features

- Profile, About, Experience, Projects and Contact sections
- Responsive design with dedicated media queries (`mediaqueries.css`)
- Interactive navigation (hamburger menu) powered by plain JavaScript
- Downloadable CV directly from the page

## Technologies

- HTML5 (semantic structure)
- CSS3 (custom properties, flexbox, media queries)
- JavaScript (DOM manipulation, no dependencies)

## Usage

No installation required:

```bash
git clone https://github.com/Abbassimedayoub/Portfolio.git
cd Portfolio
# open index.html in your browser
```

## Skills Demonstrated

Responsive web design without a framework, semantic HTML, CSS architecture (separate stylesheet for media queries), and vanilla JavaScript DOM interaction.
